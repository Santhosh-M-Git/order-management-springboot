# order-management-springboot
REST-based Order Management System using Spring Boot

# Order Management System

## Technologies Used
- Java
- Spring Boot
- REST API
- Oracle SQL

## Features
- Create Order
- View Orders
- Update Order
- Delete Order

## Database
Oracle SQL

## API Endpoints
POST   /api/orders
GET    /api/orders
GET    /api/orders/{id}
PUT    /api/orders/{id}
DELETE /api/orders/{id}

## How to Run
1. Clone the repository
2. Import into STS
3. Run as Spring Boot App
